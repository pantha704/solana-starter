# Solana Starter

Complete Solana development starter kit with Rust and TypeScript implementations covering NFT creation, SPL tokens, and vault operations.

## Overview
This project serves as a comprehensive starting point for Solana development, providing both Rust and TypeScript implementations for various Solana functionalities.

## Features
- NFT creation and management
- SPL token operations
- Vault operations
- Cross-platform client implementations

## Technologies
- Rust
- TypeScript
- Anchor
- Solana CLI

## Getting Started
1. Install Solana CLI tools
2. Install Anchor CLI
3. Run `anchor build` to build the programs
4. Run `anchor test` to run tests
